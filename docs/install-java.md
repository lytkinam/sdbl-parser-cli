# Установка Java 17 (JDK) в Windows

## Ссылка для скачивания

**Сайт:** https://jdk.java.net/java-se-ri/17-MR1

Это официальный сайт **Reference Implementation (RI)** от Oracle/OpenJDK — эталонная, полностью бесплатная реализация Java 17.

## Пошаговая установка

### 1. Скачивание

1. Откройте ссылку: https://jdk.java.net/java-se-ri/17-MR1
2. В разделе **Windows** нажмите на ссылку с расширением `.zip` (например, `openjdk-17+35_windows-x64_bin.zip`).
3. Сохраните архив в папку `Загрузки`.

### 2. Распаковка

1. Откройте папку `Загрузки`.
2. Распакуйте скачанный `.zip` архив.
3. Переместите распакованную папку (например, `jdk-17`) в удобное место, например:
   ```
   C:\Program Files\Java\jdk-17
   ```

### 3. Настройка переменных среды

1. Нажмите **Win + R**, введите `sysdm.cpl` и нажмите **Enter**.
2. Перейдите на вкладку **«Дополнительно»** → нажмите **«Переменные среды»**.
3. В блоке **«Системные переменные»** нажмите **«Создать»**:
   - **Имя переменной:** `JAVA_HOME`
   - **Значение переменной:** `C:\Program Files\Java\jdk-17` (укажите свой путь)
4. Найдите переменную `Path` в списке системных переменных, выделите её и нажмите **«Изменить»**.
5. Нажмите **«Создать»** и добавьте строку:
   ```
   %JAVA_HOME%\bin
   ```
6. Нажмите **«ОК»** во всех окнах.

### 4. Проверка установки

Откройте **Git Bash** (или `cmd`) и выполните:

```bash
java -version
```

Ожидаемый вывод:
```
openjdk version "17" 2021-09-14
OpenJDK Runtime Environment (build 17+35-2724)
OpenJDK 64-Bit Server VM (build 17+35-2724, mixed mode, sharing)
```

Если версия отображается — установка завершена успешно.

## Альтернативы

Если по какой-либо причине данный сайт недоступен, подойдёт любой другой JDK 17+:
- Oracle JDK
- Amazon Corretto
- BellSoft Liberica JDK
- Azul Zulu

Главное требование: команда `java -version` должна выводить версию **17** или выше.
